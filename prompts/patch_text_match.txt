# Hard Matching Rules (核心对齐规则)
请严格遵循以下算法级规则执行文本检索：

1. **绝对逐字还原 (100% Verbatim)**：你返回的文本必须是上面《目标 Section 实际内容》中**原样存在、连续不断**的子字符串。严禁修改、增删、替换任何一个字符、空格或标点符号。
2. **逆向意译解析 (De-paraphrasing)**：理解输入的"意图文本"可能是原文的意译版、缩略版或语义混合版。请看破其核心语义，强行映射回实际内容中对应的最吻合段落。
3. **边界收拢与聚焦**：如果意图文本的描述较为宽泛并模糊指向了多处原文，请仅截取、返回**最核心、最长、最具实质意义**的那一段连续文本。
4. **彻底熔断机制**：如果遍历全文后，确认意图文本在实际内容中完全找不到任何语义相似的对齐项，**请保持输出绝对空白（什么都不要输出）**，严禁编造或输出任何提示语。

---

# Output Format
**【极度严苛约束】**：
- 仅输出匹配到的实际原文。
- **严禁**添加任何 Markdown 代码块包裹（如 ``` ）。
- **严禁**包含任何解释、前言、后缀或标点符号。
- 如果无法匹配触发熔断，请保持输出完全为空（零字符输出）。
